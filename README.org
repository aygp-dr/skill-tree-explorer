#+TITLE: skill-tree-explorer
#+AUTHOR: aygp-dr

Interactive skill tree visualization and progress tracking. Built with Flask for Replit deployment.

** Run locally

#+begin_src bash
pip install -r requirements.txt
python main.py
#+end_src

** Deploy on Replit

1. Fork this repo on Replit
2. It auto-detects the =.replit= config
3. Click Run

** Features

- 3 skill trees: Backend, Frontend, DevOps
- Dependency-based skill unlocking
- Progress tracking with SQLite
- Visual progress bar per tree
- JSON API at =/api/trees= and =/api/tree/<id>=

** Stack

- Python 3.11+
- Flask
- SQLite (auto-created in =data/=)
- Vanilla CSS (dark theme, no frameworks)
