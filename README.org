* Skill Tree Explorer

Interactive skill tree visualization and progress tracking. Built with Flask.

[[https://github.com/aygp-dr/skill-tree-explorer/actions/workflows/ci.yml][https://github.com/aygp-dr/skill-tree-explorer/actions/workflows/ci.yml/badge.svg]]

** Features

- *3 skill trees*: Backend Engineering, Frontend Engineering, DevOps & Infrastructure
- *Dependency-based unlocking*: skills lock until prerequisites are completed
- *Progress tracking*: per-tree completion percentage with SQLite persistence
- *Visual progress bar*: dark-themed UI with level-based skill layout
- *JSON API*: ~/api/trees~ and ~/api/tree/<id>~ endpoints
- *Replit-ready*: auto-detected ~.replit~ configuration

** Quick Start

#+begin_src bash
pip install -r requirements.txt
make run
#+end_src

The app runs on ~http://localhost:8080~ by default.

** API

| Endpoint             | Method | Description                          |
|----------------------+--------+--------------------------------------|
| ~/~                  | GET    | Main UI with skill tree visualization |
| ~/~                  | POST   | Toggle skill completion              |
| ~/api/trees~         | GET    | List all trees (id -> name)          |
| ~/api/tree/<id>~     | GET    | Tree details with completion status  |
